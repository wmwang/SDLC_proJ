## 為什麼

React App 目前僅有鬧鐘功能，缺少即時資訊展示能力。新增全球天氣資訊功能，讓使用者能一目了然查看世界各國主要城市的天氣狀況（溫度、天氣描述、濕度、風速等），提升應用實用性與使用者體驗。

## 變更內容

- 新增全球天氣資訊頁面，展示世界各國主要城市的即時天氣
- 整合 Open-Meteo 免費天氣 API（無需 API Key），取得即時天氣資料
- 提供城市搜尋與篩選功能，方便快速查找特定城市天氣
- 支援溫度單位切換（攝氏 / 華氏）
- 以卡片式 UI 呈現各城市天氣資訊，包含天氣圖示、溫度、濕度、風速等
- 依洲別/區域分類顯示城市群組

## 能力範圍

### 新增能力
- `weather-display`: 天氣資訊展示核心功能 — 呼叫天氣 API 取得資料並以卡片元件呈現
- `city-search`: 城市搜尋與篩選 — 依名稱搜尋、依區域篩選城市列表
- `temperature-unit`: 溫度單位切換 — 攝氏與華氏即時轉換

### 修改能力

（無既有能力需修改）

## 影響範圍

- **前端元件**：新增 WeatherPage、WeatherCard、CitySearchBar 等元件
- **新增依賴**：使用瀏覽器原生 `fetch` 呼叫 Open-Meteo API，不新增外部套件
- **路由**：需引入輕量路由機制（hash-based tab 切換）以在不同功能頁面間導航
- **CSS**：新增天氣頁面相關樣式
