## 為什麼

React App 目前缺少鬧鐘功能，使用者無法設定鬧鐘提醒。隨著應用程式功能增加，時間管理是常見需求，現在是加入此功能的好時機。

## 變更內容

- 新增鬧鐘設定功能（時間選擇、標籤、鬧鐘音效）
- 新增鬧鐘列表顯示（顯示所有已設定的鬧鐘）
- 支援鬧鐘啟用/停用切換
- 支援刪除鬧鐘
- 鬧鐘觸發時顯示通知或鬧鐘介面

## 能力範圍

### 新增能力
- `alarm-clock`: 鬧鐘功能（設定鬧鐘時間與標籤、鬧鐘列表管理）

### 修改能力
（無）

## 影響範圍

- `react-app/src/App.tsx`：主要 UI 元件
- `react-app/src/components/`：新增鬧鐘相關元件
- `react-app/src/hooks/`：新增鬧鐘邏輯 hooks
- `react-app/src/types/`：新增鬧鐘類型定義
