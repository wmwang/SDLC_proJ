## 架構師審查（proposal.md, design.md, plan.md）

### 發現

- **解決背景降頻（Throttling）與休眠喚醒**：將檢查機制重構為「真實 Date 區間比對（遍歷起訖時間跨越的所有日期）」，有效防止跨日或大於 24 小時休眠導致的漏響 Bug。
- **定時器效能優化**：在 `useAlarmChecker` 中引入了 `alarmsRef` 與 `onTriggerRef`，將定時器 `useEffect` 依賴項設為空陣列 `[]`，防止因 CRUD 頻繁重置/銷毀計時器。
- **解鎖自動播放與聚焦**：在通知 `onclick` 事件上綁定 `window.focus()`，當使用者點選桌面通知時會引導其返回該 React 頁面，且在 `play().catch()` 中加入 UI 自動播放受阻提示。
- **測試不穩定度修正**：在 `useAlarmChecker.test.js` 測試中，顯式 mock 屬性 `Object.defineProperty(document, 'visibilityState', { value: 'visible', configurable: true })`，保障了測試的可靠性。
- **文件一致性**：移成了不必要引入且會增加複雜度的 Web Worker 降級文字。
